💻 Calculadora Web Simples
Este é um projeto de Calculadora Funcional desenvolvido para consolidar conhecimentos fundamentais de lógica de programação e manipulação de eventos no DOM (Document Object Model).

O foco principal foi a implementação de operações aritméticas básicas de forma limpa e eficiente.

🚀 Tecnologias Utilizadas
HTML5: Estruturação semântica da interface.

CSS3: Estilização moderna e layout responsivo.

JavaScript (Vanilla): Lógica das operações e controle de estado.

🧮 Funcionalidades
A calculadora permite realizar as quatro operações fundamentais da aritmética:

Somar valores.

Subtrair valores.

Multiplicar valores.

Dividir valores.
